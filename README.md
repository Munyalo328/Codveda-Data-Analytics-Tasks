# Codveda Data Analytics Projects

## Overview

This repository contains projects completed as part of the **Codveda Data Analytics Program**. The projects are organized into three levels: Basic, Intermediate, and Advanced. Each project focuses on developing practical data analytics skills using Python and industry-standard tools for data cleaning, exploration, visualization, machine learning, time series analysis, clustering, and dashboard development.

## Objectives

The primary objectives of these projects are to:

* Develop data cleaning and preprocessing skills.
* Perform exploratory data analysis (EDA).
* Create effective data visualizations.
* Build and evaluate machine learning models.
* Analyze time series data.
* Apply clustering techniques for segmentation.
* Design interactive business intelligence dashboards.
